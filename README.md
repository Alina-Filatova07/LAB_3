# LAB_3
Лабораторная работа: Калькулятор математических выражений с производными.
Удалить все скомпилированные файлы: make clean
Скомпилировать программу и тесты: make all
Запустить основную программу: .\lab
Запустить тесты: .\test_runner
После выполнения программы создадутся 2 файла - lab.exe и test_runner.exe
